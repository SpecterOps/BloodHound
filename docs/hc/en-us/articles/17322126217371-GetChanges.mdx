## Abuse Info

This edge is not abuseable by itself.

When combined with [GetChangesAll](/hc/en-us/articles/17362283577627-GetChangesAll), BloodHound will create the abuseable edge [DCSync](/hc/en-us/articles/17322385609371-DCSync).

When combined with [GetChangesInFilteredSet](/hc/en-us/articles/17362383096219-GetChangesInFilteredSet), BloodHound will create the abuseable edge [SyncLAPSPassword](/hc/en-us/articles/17358058147355-SyncLAPSPassword).

## Opsec Considerations

This edge has no opsec considerations.
