The principal is allowed to synchronize (DCSync) the Filtered Attribute Set (FAS), which are the attributes not replicated to RODCs.

## Abuse Info

This edge is not abuseable by itself. When combined with [GetChanges](/hc/en-us/articles/17322126217371-GetChanges), BloodHound will create the abuseable edge [SyncLAPSPassword](/hc/en-us/articles/17358058147355-SyncLAPSPassword).

## Opsec considerations

This edge has no opsec considerations.
