For general instructions on adding a SAML provider to BloodHound Enterprise, or for configuring users to utilize a SAML provider, see [SAML in BloodHound Enterprise](/hc/en-us/articles/9228122981275-SAML-in-BloodHound).

See [SAML Order of Operations and Quick Reference](/hc/en-us/articles/9228122981275-SAML-in-BloodHound) before starting.

## Create an Auth0 Application

1.  Create an authentication application for BloodHound Enterprise in Auth0.
 <Frame>
  <img src="/hc/images/image-2-39.png"/>
</Frame>
2.  Assign a recognizable name to the application and select the integration option for "Regular Web Application."
<Frame>
  <img src="/hc/images/image-2-40.png"/>
</Frame>
3.  After the creation of the application in Auth0 you should see the application details page.
<Frame>
  <img src="/hc/images/image-2-41.png"/>
</Frame>
4.  Click on the "Addons" tab.
<Frame>
  <img src="/hc/images/image-2-42.png"/>
</Frame>
5.  Enable the "SAML2 Webapp" toggle and download the Identify Provider Metadata file.
<Frame>
  <img src="/hc/images/image-2-43.png"/>
</Frame>
6.  Follow the instructions at [SAML in BloodHound Enterprise](/hc/en-us/articles/9228122981275-SAML-in-BloodHound) to create the SAML provider in BloodHound Enterprise.
